# 02 — Glossary

This glossary collects domain terms used across the Roundhouse documentation.

- Collection — The set of models and sets owned by a collector.
- Collection Item — A purchased instance of a railway model or set (a purchase record).
- Railway Model (ModelProduct) — A manufacturer product (locomotive, car, set). Identified by manufacturer + product code.
- Rolling Stock — Individual vehicles included in a Railway Model (locomotive, coach, wagon, EMU car).
- OwnedRollingStock — (concept) a physical unit owned by a collector, derived from a RollingStock template.
- Consist — A train formation: an ordered list of rolling stock representing a train at a point in time.
- Maintenance Card — Per-unit maintenance and digital setup record (last/next maintenance, decoder info, events).
- Seller — A shop or private seller (shop/collector) from which items are purchased.
- URN — Uniform Resource Name used as canonical identifier for primary entities (see entity dictionary).
- Seed data — Bundled JSON resources (scales, railway companies, manufacturers) shipped with the app.

Acronyms
- KMP — Kotlin Multiplatform
- UI — User Interface
- DCC — Digital Command Control (model railway digital control standard)
- JSON — JavaScript Object Notation
- CSV — Comma-Separated Values
