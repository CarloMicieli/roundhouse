This file lists primary user stories and use cases.

### Initial Collection Setup

- As a new user, I want to create an initial collection when I first start the app so I can begin tracking my owned items quickly.

## Collection Management

- As a collector, I want to add models and sets to my collection so I can keep track of what I own and their purchase details.
- As a collector, I want to edit or remove items from my collection so the collection stays accurate.
- As a collector, I want to search and filter my collection by manufacturer, category, scale, or price so I can quickly find specific items.

## Wish List Management

- As a collector, I want to create and manage wish lists of railway models and sets so I can plan future purchases.
- As a collector, I want to move items from a wish list to my collection so I can record acquisitions.

## Rolling Stock & Railway Model Management

- As a collector, I want to add/edit/remove railway models and their rolling stock so my catalog is accurate.

## Depot Management

- As a collector, I want to view all owned rolling stock in one place so I can manage my inventory.
- As a collector, I want to see technical and maintenance info for each rolling stock item so I can plan maintenance or upgrades.

## Maintenance

- As a collector, I want to view and edit maintenance status and history per owned unit so I can track upkeep.
- As a collector, I want reminders when maintenance is due so I don't miss upkeep tasks.

## Consists

- As a collector, I want to create and manage consist sheets so I can document train formations.
- As a collector, I want to add, remove and reorder cars in a consist so the formation is accurate.
